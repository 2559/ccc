# SMT 订单跟踪小程序

基于原生微信小程序的 SMT（表面贴装技术）生产订单进度跟踪系统。

## 功能特点

- 📋 **订单列表**：卡片式展示所有订单，支持按状态筛选
- 🔍 **订单详情**：查看完整订单信息和状态时间线
- ➕ **新增订单**：快速创建新订单，自动生成订单号
- 🔄 **状态管理**：支持订单状态流转（待生产 → 生产中 → 已完成 → 已发货）

## 技术栈

- 原生微信小程序框架（JavaScript）
- 本地缓存存储（wx.setStorageSync）
- 简洁工业风格的 UI 设计

## 项目结构

```
smt-order-tracker/
├── app.js              # 应用入口
├── app.json            # 应用配置
├── app.wxss            # 全局样式
├── sitemap.json        # 站点地图
├── assets/             # 静态资源
└── pages/
    ├── list/            # 订单列表页
    ├── detail/          # 订单详情页
    └── create/          # 新增订单页
```

## 使用方法

1. 克隆项目到本地
2. 使用微信开发者工具导入项目
3. 选择项目目录
4. 使用测试号或正式 AppID 运行

## 订单状态

| 状态 | 颜色 | 说明 |
|------|------|------|
| 待生产 | 橙色 | 订单已创建，等待开始生产 |
| 生产中 | 蓝色 | 正在生产中 |
| 已完成 | 绿色 | 生产已完成，等待发货 |
| 已发货 | 灰色 | 已完成发货 |

## 数据存储

所有数据存储在本地缓存中，换机或清除缓存后数据将丢失。如需持久化，请自行对接后端 API。
